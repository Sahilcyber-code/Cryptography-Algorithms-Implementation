Text Encryption Project

Secure AES, DES and RSA encryption with comparison and safe key handling.

Keys and secrets are never stored by the server; only operation metadata is logged.

Symmetric Encryption

Secure key handling: Use strong passphrases, avoid reusing IVs, and prefer AES over DES for sensitive data.

RSA Encryption

RSA usage: Use the public key for encryption and the private key for decryption. Never share your private key.

Hybrid Encryption (AES + RSA)

Hybrid: AES encrypts the message; RSA encrypts the AES key. Keep private keys secret.

Algorithm Comparison

Algorithm Type Strength Use Case Notes

Use AES for modern symmetric encryption. DES is included for legacy compatibility, but it is not secure for production. RSA is ideal for key exchange and asymmetric encryption.
